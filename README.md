FarmPulse – Crop Marketplace System

FarmPulse is a modern, responsive frontend web application designed to simulate a digital marketplace that connects farmers (sellers) and users (buyers). The application focuses on providing a clean, user-friendly interface where farmers can manage crop listings and users can easily browse and explore available crops.

This project demonstrates real-world application flow using frontend technologies only, with well-structured UI components, smooth navigation, and interactive features. It is built to showcase how an agricultural marketplace system can function efficiently in a digital environment.

🚀 Features
👨‍🌾 Farmer Features
Dashboard interface to manage crop listings
Add new crops with details like name, price, quantity, village, and contact
Edit and delete crop entries (UI level)
View all added crops in a structured layout
👤 User Features
Browse all available crops in a card-based layout
View crop details including price, quantity, and location
Contact farmer using provided details
Smooth and simple browsing experience
🔍 Search & Filter
Search crops using keywords (case-insensitive)
Filter crops based on village, price, and crop type
Sort crops (low to high / high to low)
Display message when no crops are available
🎨 UI/UX Features
Agriculture-themed design with green color palette
Responsive layout (mobile + desktop)
Smooth navigation and transitions
Interactive buttons and hover effects
Loading states and clean form design
🛠️ Technologies Used
HTML – Structure of web pages
CSS – Styling and responsive layout
JavaScript – Interactivity and logic
React.js – Component-based UI development
Bootstrap – Responsive design and UI components
📁 Project Structure
Components – Reusable UI elements (Navbar, Cards, Forms)
Pages – Home, Login, Signup, Dashboard, Crop Listing
Styles – CSS files for layout and design
Assets – Images and icons used in UI

The project follows a modular structure, making it easy to maintain and extend in future.

🔄 Application Flow
User opens the application
Selects role (Farmer/User)
Navigates to login or signup page
After login (UI level), user is redirected
Farmer manages crops through dashboard
User browses, searches, and views crop details
💾 Data Handling

This is a frontend-only project.

Uses static/mock data for demonstration
Can optionally use localStorage for temporary data persistence
No backend or database integration
⚡ Performance & Stability
Smooth UI without freezing
Proper input validation
No blank or broken screens
Optimized for fast loading
Clean and structured code
🔮 Future Scope
Backend integration with database
Real-time data updates

Mobile application development

🌐 Deployment

The project can be deployed using platforms like Vercel or Netlify for public access.

📌 Description

FarmPulse is a frontend prototype that simulates a real-world agricultural marketplace system. It focuses on UI/UX design, application flow, and user interaction while demonstrating how farmers and buyers can connect through a digital platform.
